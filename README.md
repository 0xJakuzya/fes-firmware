# Прошивка устройства функциональной электрической стимуляции (ФЭС)

Встроенное ПО микроконтроллерного устройства для нейромышечной электростимуляции. Устройство формирует симметричные биполярные импульсы тока по 8 независимым каналам и управляется дистанционно по Wi-Fi/Ethernet через бинарный TCP-протокол.

Область применения: медицинская реабилитация, нейрофизиологические исследования, роботизированные реабилитационные комплексы.

## Аппаратный состав

| Компонент | Назначение |
|-----------|-----------|
| ESP32 / ESP32-S3 / ESP32 Eth01| Основной микроконтроллер |
| PCA9685 | 16-канальный ШИМ-генератор |
| DRV8871 × 8 | H-мосты для формирования биполярного тока |

Каждый канал стимуляции управляется парой выходов PCA9685 через H-мост. Конденсатор 1 мкФ/63 В в цепи обеспечивает перераспределение заряда в межимпульсной паузе.

## Параметры стимуляции

| Параметр | Диапазон |
|----------|----------|
| Амплитуда | 0–255 |
| Частота | 1–100 Гц |
| Ширина импульса | 100–15 000 мкс |

Форма сигнала — пятифазный биполярный импульс. При превышении суммарной длительности активных фаз над периодом ширина импульса автоматически ограничивается.

## Сетевой интерфейс

Устройство поднимает точку доступа Wi-Fi и принимает TCP-соединения. Параллельно поддерживается проводной Ethernet. Пароль точки доступа можно сменить по сети.

Протокол — бинарный, с трёхбайтовым заголовком (тип сообщения, идентификатор запроса, длина полезной нагрузки). Поддерживаемые команды: получение информации об устройстве, смена пароля Wi-Fi, установка и чтение параметров канала, поканальный старт и стоп стимуляции. Каждый канал имеет состояние — отключён, готов или активен.

## Сборка и прошивка

Требования: ESP-IDF ≥ v5.5.3, CMake ≥ 3.16.

```bash
idf.py set-target esp32s3
idf.py build
idf.py -p COM1 flash monitor
```
